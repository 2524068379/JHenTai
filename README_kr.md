기기 ABI에 맞는 APK를 설치하세요.

- `arm64-v8a`: 대부분의 최신 Android 기기
- `armeabi-v7a`: 구형 ARM 기기
- `x86_64`: 주로 에뮬레이터

업데이트는 새 APK를 기존 앱 위에 설치하면 됩니다.

## 주요 기능

- 온라인 열람과 갤러리 다운로드
- 아카이브 다운로드, 압축 해제, 열람
- 모바일 검색, 태그 제안, 페이지 점프, 이미지 검색
- 즐겨찾기, 평점, 토렌트, 통계, 댓글, 공유
- 비밀번호 로그인, 쿠키 로그인, Android WebView 로그인
- 태그 번역, 태그 투표, 관심 태그, 차단 규칙
- 로컬 갤러리 열람과 Android 공유 인텐트 지원
